# TextBatch Pro - Batch Text Processing Tool

Batch process text files: find/replace, deduplicate, trim, stats, extract links.

## Features
- Batch find and replace across files
- Remove empty lines and trim whitespace
- Deduplicate lines
- Text statistics (words, chars, lines, Chinese chars)
- Extract URLs and emails
